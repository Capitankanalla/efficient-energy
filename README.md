📘 README — Projecte Web d’Assessorament Energètic
🧭 Objectiu del projecte
Crear una web moderna, clara i professional per oferir serveis d’assessorament energètic, amb un disseny net, contingut útil i funcionalitats que ajudin l’usuari a contactar, comparar factures i entendre com pot estalviar energia.

Aquesta web ha de transmetre:

Confiança

Professionalitat

Transparència

Facilitat d’ús

🏗️ Estructura general del projecte
La web es construirà en fases, començant per una versió estàtica i evolucionant cap a funcionalitats dinàmiques.

Fase 1 — Versió estàtica (Frontend pur)
Header amb logo i menú

Hero amb titular, subtítol i botó d’acció

Secció de serveis (3 blocs)

Secció “Per què confiar en mi?”

Secció de contacte (formulari visual)

Footer amb dades i enllaços

Disseny responsiu

Fase 2 — Millores de disseny
Paleta de colors coherent

Tipografia moderna

Espaiats i jerarquia visual

Imatges optimitzades

Menú lateral o superior segons evolució

Fase 3 — Funcionalitats
Formulari de contacte funcional (backend)

Formulari de comparació de factura

Carrega de dades dinàmica

Multiidioma (CAT / ES / EN)

Àrea privada (opcional, futur)

Fase 4 — Backend
API pròpia (Node.js)

Enviament de correus

Validació de dades

Integració amb serveis externs

Desplegament del backend (Render / Railway)
